# 원클릭업무포털

로그인된 Edge 세션을 이용해 나이스와 K-에듀파인의 업무 입력 화면까지 이동하는 Windows 데스크톱 프로그램입니다.

## 설치

GitHub의 **Releases**에서 최신 `Setup.exe`를 내려받아 설치합니다.

설치형 프로그램은 시작할 때 GitHub Release의 새 버전을 확인합니다. 새 버전이 있으면 자동으로 다운로드하고 적용한 뒤 다시 실행합니다.

## 사용 순서

1. 프로그램에서 `로그인`을 누릅니다.
2. 열린 Edge에서 업무포털 로그인과 인증서 인증을 직접 완료합니다.
3. 프로그램에서 해당 Edge 창을 선택하고 `연결`을 누릅니다.
4. 프로그램이 업무포털에서 `나이스`와 `에듀파인` 항목을 찾아 미리 열고 브라우저를 최소화합니다. 항목을 찾지 못하면 선택한 교육청의 공식 주소로 새 탭을 엽니다.
   - 업무포털 공지창이 열려 있으면 `1주일동안 열지 않기`를 선택하고 자동으로 닫습니다.
   - K-에듀파인에 처음 접속할 때 필요한 로컬 네트워크 접근 권한을 자동으로 부여하고, 전용 Edge 프로필에서 선택한 교육청 공식 주소의 WXSClient 실행을 허용합니다.
5. `나이스`, `복무`, `출장`, `에듀파인`, `기안`, `품의` 버튼으로 필요한 화면을 엽니다.
6. 설정에서 세션 자동 연장을 켜면 나이스와 K-에듀파인의 남은 시간이 20분 이하일 때만 공식 연장 버튼을 실행합니다. 페이지 자체는 자동으로 새로고침하지 않습니다.
7. 다른 지역 사용자는 설정에서 소속 교육청을 선택한 뒤 새 로그인 창을 엽니다.
8. 내용을 확인하고 입력한 뒤 최종 상신 또는 결재 요청은 사용자가 직접 실행합니다.

## 지역 및 창 설정

- 서울·경기·경남·부산·대구·대전·경북·세종·울산·인천·광주·전남·전북·충남·충북·강원·제주를 지원합니다.
- 선택한 교육청에 맞춰 업무포털, 나이스, K-에듀파인 주소를 자동으로 사용합니다.
- 기존 사용자와 설정값이 없는 사용자의 기본 교육청은 전북입니다.
- `프로그램을 항상 위에 표시`는 기본적으로 꺼져 있으며 설정에서 켜거나 끌 수 있습니다.

## 안전 원칙

- 로그인과 인증서 선택은 사용자가 직접 수행합니다.
- 프로그램은 최종 상신·결재 요청 버튼을 자동으로 누르지 않습니다.
- 브라우저 권한 설정은 원클릭 업무포털의 전용 Edge 프로필 안에서 선택한 교육청의 K-에듀파인 주소와 `wxsclient` 프로토콜에만 적용합니다.
- 로그인 정보, 인증서 암호, 개인정보와 세션 토큰을 저장하지 않습니다.
- 사용자 수 집계는 로그인 없이 익명 설치 식별자의 SHA-256 해시와 앱 버전만 전송합니다.
- 기본 사용 통계는 하루 한 번, 현재 사용자 신호는 한국 시간 오전 7시~11시에만 30분 간격으로 전송합니다.
- 이름, 이메일, 학교, 나이스·K-에듀파인 화면 내용은 사용 통계에 포함하지 않습니다.
- 익명 사용 통계는 설정에서 언제든지 끌 수 있습니다.

## Firebase 사용 통계

Firebase 프로젝트 ID는 `oneclickportal`입니다. Cloud Function은 새 설치 수와 날짜별 활성 설치 수를 Cloud Firestore에 기록합니다.

설정창에는 항상 `현재 n명의 사용자가 원클릭 업무포털을 사용 중입니다`로 표시합니다. 내부 집계는 오전 7시~11시 동안 최근 40분 인원을 사용하고, 그 밖의 시간에는 가장 최근 오전 11시 집계값을 유지합니다.

- 전체 설치 수: `usageStats/global.totalInstallations`
- 날짜별 활성 설치 수: `usageDaily/{yyyy-MM-dd}.activeInstallations`
- 익명 설치별 최초·최근 실행: `anonymousInstalls/{SHA-256 해시}`

로그인 없는 집계이므로 한 사람이 여러 PC에 설치하거나 레지스트리를 초기화하면 별도 설치로 계산됩니다. 또한 공개 HTTP 엔드포인트 특성상 이 수치는 정확한 개인 수가 아니라 활용 추세를 보는 추정치입니다.

백엔드 테스트와 배포:

```powershell
cd .\functions
npm install
npm test
cd ..
npx firebase-tools deploy --only firestore:rules,functions
```
